## How to run

### Installation
in folders server and client

```bash
$ npm install
```

### Running the app
in root of project
```bash
# development in watch mode
$ docker-compose -f docker-compose.dev.yml up --build

# production
$ docker-compose -f docker-compose.prod.yml up --build
```
